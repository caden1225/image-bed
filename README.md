# image-bed

Personal image bed hosted on GitHub Pages.

## URL convention

- Raw file: `https://raw.githubusercontent.com/caden1225/image-bed/main/<path>`
- Pages URL: `https://caden1225.github.io/image-bed/<path>`

For public image hotlinking, prefer the **Pages URL** (CORS-friendly).

## Upload

Use the helper script:

```bash
./upload.sh path/to/image.png [subdir/]
```

Or manually place images in this repo and push.

## Size limits

- Keep files under 5 MB when possible.
- GitHub blocks single files larger than 100 MB.
